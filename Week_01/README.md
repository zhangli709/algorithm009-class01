# 时间复杂度和空间复杂度
## 时间复杂度总共分为七种
* 常数复杂度
* 对数复杂度
* 线性复杂度
* n^2复杂度
* n^3复杂度
* 指数复杂度
* 阶乘复杂度
## 空间复杂度
* 一般就常数复杂度
* 线性复杂度。
