# 动态规划

## 动态规划复习： 附带递归， 分治

    1. 递归：自己调用自己
    2. 分治： 多种子情况的递归。 人肉递归低效，找规律，找到最近最简方法，将其拆解为可重复解决的问题。数学归纳法。
    3. 分治过程中淘汰不符合的解，就是动态规划。=分治 + 最优子结构   ---> 动态递推。
